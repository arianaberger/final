## Quick-Start Instructions

```bash
# Clone this repository
$ git clone git@github.com:arianaberger/savvy-saver.git

# Go into the repository
$ cd savvy-saver

# Install dependencies
$ npm install

# Run the app by opening two tabs
# Tab 1
$ rails s -p 3001

# Tab 2
$ cd client
$ npm start
```
